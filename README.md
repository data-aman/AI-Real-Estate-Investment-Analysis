# AI-Powered Real Estate Investment Analysis

An automated workflow that screens residential investment properties using predefined investment criteria and generates structured investment reports.

## Project Overview

This project automates the first stage of real estate investment analysis by:

- Collecting property listing data
- Evaluating investment metrics
- Calculating cash flow, Cap Rate, and Cash-on-Cash Return
- Identifying qualified investment opportunities
- Generating professional investment reports
